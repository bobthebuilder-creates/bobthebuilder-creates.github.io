  • DNS Anomaly Detection Pipeline

    ML Engineering · 2023 – 2024

    Problem

    High-volume DNS telemetry in classified operational environments. Rule-based detection creates alert fatigue; labeled training data doesn’t exist for novel adversary tradecraft.

    Technical approach

    Unsupervised anomaly detection combining a temporal convolutional network (TCN) autoencoder with HDBSCAN clustering. The TCN learns normal DNS behavior from unlabeled telemetry; HDBSCAN identifies anomalous behavioral clusters without requiring labeled data or predefined signatures.

    Outcome

    Deployed in operational environment. Surfaces anomalous DNS clusters for analyst review rather than generating raw alerts — reducing triage burden while preserving analyst judgment in the loop.

    Python TCN Autoencoder HDBSCAN Anomaly Detection Unsupervised ML

  • NeuroTrade-X1

    R&D · Quantitative Finance · 2024

    Problem

    Quantitative strategy evaluation requires vast simulation. Conventional ML produces black-box models that cannot satisfy interpretability requirements in regulated financial environments.

    Technical approach

    Evolutionary trading research system using NEAT (neuroevolution of augmenting topologies) for strategy topology evolution, distributed simulation for evaluation at scale, and LLM-based symbolic regression for post-hoc auditability — extracting human-readable decision rules from evolved neural networks.

    Outcome

    Research system demonstrating auditability techniques applicable to black-box financial models — directly transferable to AI governance problems in regulated financial services.

    Python NEAT Neuroevolution Symbolic Regression FinTech
